4/4 taktimoot – täpselt nagu rakenduses (Leland SMuFL numbrid)

Numbrid on Leland-fondi SMuFL taktimõõdu glyph'id (U+E080–E089). Kui fond laeb, näed sama kujundit mis rakenduses.

Fond: public/fonts/Leland.woff2. Kui numbrid on ruudud või tühjad, ava see leht dev-serveri kaudu (nt npm run dev ja seejärel ava docs/taktimoot-4-4-leland.html vastavalt serveri aadressile).
